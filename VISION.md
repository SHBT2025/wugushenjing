# VISION — 产品理念

> **帮助散户克服情绪交易。**
>
> 不是预测股价。
> 不是高频交易。
> 不是追涨杀跌。

---

## 核心思想

```
┌─────────────────────────────────────┐
│         心 中 无 股                 │
│                                     │
│  按策略执行 -> 减少操作次数         │
│                     ↓               │
│               减少情绪干扰          │
└─────────────────────────────────────┘
```

### 心中无股

不执著于某一只股票的涨跌，不被持仓盈亏左右判断。把注意力从
"我的股票"转移到"我的策略"上。

### 按策略执行

每一次买卖都必须有明确的规则依据，而非"感觉要涨"或"怕它跌"。
规则在前，执行在后。

### 减少操作次数

交易频率越高，情绪干扰越大，错误率越高。等待信号，而非创造信号。

### 减少情绪干扰

贪婪和恐惧是散户最大的敌人。用规则代替直觉，用系统代替冲动。

---

## 设计原则

| 原则 | 说明 |
|------|------|
| **信号驱动** | 所有操作由 RSI 指标信号触发，拒绝主观判断 |
| **一目了然** | 看板式 UI，3 秒内看清持仓状态和信号 |
| **离线可用** | PWA 本地缓存，行情数据断网也能回顾 |
| **轻量纯粹** | 单 HTML 文件，无依赖，无后台，无跟踪 |

---

## 不是什么

| ❌ 不是 | ✅ 是 |
|---------|-------|
| 预测股价的工具 | 执行策略的工具 |
| 高频交易的平台 | 减少操作的提醒器 |
| 追涨杀跌的信号 | 逆人性冷静的信号 |
| 一夜暴富的捷径 | 纪律复利的习惯 |

---

## 目标用户画像

> **张三，35 岁，上班族，炒股 3 年。**
>
> 每天忍不住看盘十几次，追涨杀跌，频繁交易。
> 年底一算账：手续费比利润还多。
>
> 他需要的不是更多信息，而是**更少的冲动**。

---

## 一句话

> **不是让散户赚更多，而是让散户亏更少。**
>
> —— 减少错误，就是盈利。
