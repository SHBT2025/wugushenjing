# 📈 RSI Pulse — 股票RSI技术指标分析工具
地址https://shbt2025.github.io/wugushenjing/
基于东方财富API的股票RSI指标分析 PWA 应用。支持**日/周/月**三周期RSI监测，自定义提醒阈值，拖拽排序，离线可用。

---

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🔍 **股票搜索** | 输入代码或名称模糊搜索A股 |
| 📊 **三周期RSI** | 日线、周线、月线 RSI 一目了然 |
| 🔔 **RSI提醒** | 自定义超买/超卖阈值，浏览器推送通知 |
| 📋 **拖拽排序** | 自由拖动股票卡片排列顺序 |
| 📱 **PWA支持** | 可安装到桌面，像原生App一样使用 |
| 📡 **离线可用** | Service Worker缓存，无网络也能打开 |

---

## 🎮 操作说明

| 操作 | 方式 |
|------|------|
| **添加股票** | 顶部搜索框输入代码/名称 → 点击搜索结果 |
| **查看RSI** | 卡片显示日线RSI，点击「周」「月」切换周期 |
| **设置提醒** | 点击 🔔 图标 → 设置阈值 → 确认 |
| **排序卡片** | 长按卡片拖拽 |
| **删除股票** | 点击 ✕ 按钮 |
| **数据刷新** | 每次打开自动刷新，或手动刷新页面 |

---

HTML5 + CSS3 + Vanilla JS · SortableJS · Service Worker · Web Notification API · 东方财富公开API
