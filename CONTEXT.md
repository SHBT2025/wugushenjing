## 1. 已完成功能

### 1.1 核心功能（全部完成且可运行）

| 模块 | 状态 | 说明 |
|------|------|------|
| **自选股管理** | ✅ 完成 | 添加/删除/选择自选股，支持拖拽排序 (SortableJS 1.15.1)，持久化到 localStorage |
| **股票搜索** | ✅ 完成 | 东方财富 searchapi 实时搜索，支持股票代码/拼音首字母/中文名称，下拉展示 |
| **K线数据获取** | ✅ 完成 | 东方财富 push2his 接口 + 腾讯财经 fallback，日/周/月三周期，lmt=500 条，含前复权 |
| **实时行情** | ✅ 完成 | 东方财富 push2 接口，60秒定时轮询，更新价格/涨跌幅到表格 |
| **K线图绘制** | ✅ 完成 | Canvas 2D 绘制，包含 K线蜡烛图、RSI 叠加曲线、十字光标、Tooltip |
| **RSI(6) 计算** | ✅ 完成 | Wilder's Smoothing 算法，6周期 RSI，O(n) 增量计算，独立函数 calcRSI6 / getRSI6Series |
| **电池条可视化** | ✅ 完成 | 10格电池条，中轴 50，RSI>50 红色向上填充，RSI<50 绿色向下填充 |
| **事件系统** | ✅ 完成 | 买入/卖出/标记三种事件类型，关联 K线日期，表格展示，支持增删 |
| **提醒系统** | ✅ 完成 | 价格上穿/下穿、RSI 日/周/月 共 5 种提醒维度，弹窗通知，1小时冷却 |
| **数据持久化** | ✅ 完成 | 3 个 localStorage Key 分别存储自选股、事件、提醒 |
| **数据导入导出** | ✅ 完成 | JSON 格式，含文件选择器，导出含时间戳命名 |
| **PWA 支持** | ✅ 完成 | Service Worker 缓存优先，manifest.json 配置完整，可安装到桌面 |
